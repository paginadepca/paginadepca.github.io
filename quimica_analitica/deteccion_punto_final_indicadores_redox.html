Detección del punto final: indicadores rédox

  • Temario
  • Inicio

La forma $\ce{In_{(ox)}}$ y la $\ce{In_{(red)}}$ tienen diferente color. Entre ambas se establece una reacción reversible. El indicador puede ser:

- el valorante: $\ce{KMnO4}$

- el exceso de valorante se pone de manifiesto con un reactivo auxiliar con el cual interacciona produciendo un cambio de color. Ejemplos:

Almidón, $\ce{I2}$

$\ce{SCN-, Fe^3+}$

$\ce{Fe(CN)6^4-, Fe^3+}$

- aquél que tiene un $E°{}'_{\!\! \ce{In_{(ox)} / In_{(red)}}}$

$ \def\ieqarrow#1#2{ \stackrel{ \Newextarrow{\xrightharpoon}{10,10}{0x21C0} \overset{#1}{ \Rule{0pt}{0.4em}{0pt} \smash{ \xrightharpoon[\hphantom{#2}]{\hphantom{#1}} } } }{ \Rule{0pt}{1.5mu}{0pt} \Newextarrow{\xleftharpoon}{10,10}{0x21BD} \underset{#2}{ \smash{ \xleftharpoon[\hphantom{#2}]{\hphantom{#1}} } } } } \ce{In_{(ox)} \ieqarrow{\large +n e^-}{} In_{(red)}} $

color $\ce{In_{(ox)}} \neq$ color $\ce{In_{(red)}}$

El intervalo de viraje: $ \Delta E = E°{}'_{\!\! \lower 2mu {\ce{In}}} \pm \dfrac{0{,}059}{n} $

Requerimientos:

• $E°{}'_{\!\! \lower 2mu {\ce{In}}} \simeq E_{\rm P.E.}$

• cambio de color rápido y reversible.

• estabilidad frente:

$\ast$ oxidantes fuertes.

$\ast \: T$ elevadas.

$\ast \: [\ce{H+}]$ elevadas.

$\ast$ mínimas reacciones parásitas.
