$\ce{I2 / I−} \rightarrow$ es un sistema débilmente oxidante y débilmente reductor.

$\ce{I2 + 2 $e^-$ <=> 2 I-} \quad E°_{\!\! \ce{I2 / I-}} \! = \pu{0,53 v}$

Solubilidad del $\ce{I2}$ en agua $< \pu{e−3 M}$.

$\ce{I2 + I- <=> I3-} \quad K \simeq 700 \quad$ (para hacer soluble al yodo molecular).

Métodos directos:

$\ce{I2 / A_{red} -> I− / A_{ox}}$

directas $\rightarrow$ Yodimetrías

$\ce{A_{ox} / I- -> A_{red} / I2} \quad$ ($\leftarrow$ 1ª etapa)

$ \boxed{ \ce{I2 / R_{red} -> I- / R_{ox}} } \quad $ reacción volumétrica

indirectas $\rightarrow$ Yodometrías
