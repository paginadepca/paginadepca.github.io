Aplicaciones de la extracción líquido-líquido

- Se aplica mucho para extraer iones metálicos de disoluciones acuosas mediante ligandos orgánicos voluminosos que se complejan alrededor del metal formando un complejo neutro, poco soluble en agua pero más en disolvente orgánico. A esta extracción se la denomina extracción de quelatos orgánicos:

$ \begin{aligned} & \mspace{8mu} \begin{gathered} \llap{\text{equilibrio}} \rlap{\text{ heterogéneo}} \\ \uparrow \end{gathered} \\ \ce{ M_{(aq)} + $\ce{R}_{\substack{\rm (aq) \\ \rm (o)}}$ \mspace{-8mu} &<=> MR_{(o)} } \\ &\mspace{8mu} K_{\rm ex} \! = f(\ce{$K$_{MR},\! $K$_{D(MR)},\! $K$_{D(R)},\! $K$_{a(R)}}) \end{aligned} \text{p.ej.:} \enspace \ce{Ca^2+ + 2 \underset{ \substack{ \uparrow \\ \llap{\text{ox}} \text{i} \rlap{\text{na}} } }{ox} <=> Ca(ox)2} $

$ \def\ieqarrow#1#2{ \stackrel{ \Newextarrow{\xrightharpoon}{10,10}{0x21C0} \overset{#1}{ \Rule{0pt}{0.4em}{0pt} \smash{ \xrightharpoon[\hphantom{#2}]{\hphantom{#1}} } } }{ \Rule{0pt}{1.5mu}{0pt} \Newextarrow{\xleftharpoon}{10,10}{0x21BD} \underset{#2}{ \smash{ \xleftharpoon[\hphantom{#2}]{\hphantom{#1}} } } } } \begin{aligned} &\ce{MR_{(aq)} \ieqarrow{}{\large K_{\rm D}} MR_{(o)}} \end{aligned} $

- Purificación de alcaloides (purificación de sustancias orgánicas). En este caso es importante la propiedad ácido-base del analito:

$\qquad K_{\rm D} = \dfrac{[\ce{B}]_{\rm o}}{[\ce{B}]_{\rm aq}}$

$ \ce{HB+_{(aq)} <=> B_{(aq)} + H+_{(aq)}} \qquad \ce{Ka} = \dfrac{[\ce{H+}] [\ce{B}]_{\rm aq}}{[\ce{HB+}]_{\rm aq}} $

Es más práctico trabajar con la $D$:

$ \eqalign{ D &= \dfrac{\sum [\ce{B}]_{\rm o}}{\sum [\ce{B}]_{\rm aq}} = \dfrac{ \overset{ \begin{subarray}{c} \text{no} \rlap{ \text{ se consideran equilibrios concurrentes en la fase orgánica.} } \\ \downarrow \end{subarray} }{\strut [\ce{B}]}_{\rm o} }{ [\ce{HB+}]_{\rm aq} + [\ce{B}]_{\rm aq} } = \dfrac{ [\ce{B}]_{\rm o} }{ \dfrac{[\ce{H+}] [\ce{B}]_{\rm aq}}{\ce{Ka}} + [\ce{B}]_{\rm aq} } = \dfrac{ [\ce{B}]_{\rm o} }{ [\ce{B}]_{\rm aq} \left( \dfrac{[\ce{H+}]}{\ce{Ka}} + 1 \right) } = \\[1ex] &= \dfrac{K_{\rm D}}{1 + \dfrac{[\ce{H+}]}{\ce{Ka}}} } $

Representación de $\log D$ vs. $\ce{pH}$:

$ \ce{pH} \ll \ce{pKa} \Rightarrow \log D = \log (K_{\rm D} \ce{Ka}) + \ce{pH} $

$\ce{pH} \gg \ce{pKa} \Rightarrow \log D = \log K_{\rm D}$
