Equilibrio ácido-base en medio no acuoso

  • Temario
  • Inicio

El disolvente influye en el equilibrio ácido-base con:

  • $\llap{\bullet \;}$Propiedades ácidas/básicas.
  • $\llap{\bullet \;} \varepsilon$ (cte. dieléctrica). Los disolventes menos polares no favorecen la separación de cargas, es decir, la aparición de iones.
  • $\llap{\bullet \;}$Interacciones específicas: Puentes de hidrógeno, fuerzas hidrofóbicas.

Ejemplo con diferentes disolventes:

$ \Newextarrow{\xleftharpoonup}{40,40}{0x21BC} \Newextarrow{\xrightharpoonup}{40,40}{0x21C0} \begin{array}{l} \ce{ NH2CONH2 + H2O {} $\raise 2px { \underset{ \Rule{0pt}{0.2em}{0pt} \smash{\rightharpoondown} }{ \xleftharpoonup[]{} }}$ {} NH2CONH3+ + OH- } \\[1ex] \ce{ NH2CONH2 + H2O {} $\raise 2px {\underset{ \Rule{0pt}{0.2em}{0pt} \smash{\rightharpoondown} }{ \xleftharpoonup[]{} }}$ {} NH2CONH- + H3O+ } \\[1ex] \ce{ NH2CONH2 + HCOOH {} $\raise 2px {\underset{ \Rule{0pt}{0.2em}{0pt} \smash{\leftharpoondown} }{ \xrightharpoonup[]{} }}$ {} NH2CONH3+ + HCOO- } \\[1ex] \ce{ NH2CONH2 + NH3_{(l)} {} $\raise 2px {\underset{ \Rule{0pt}{0.21em}{0pt} \smash{\leftharpoondown} }{ \xrightharpoonup[]{} }}$ {} NH2CONH- + NH4+ } \end{array} $

Esto sirve para exaltar las propiedades ácidas o básicas de una especie y facilitar las valoraciones. También para mejorar la solubilidad.
