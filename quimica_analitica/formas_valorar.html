Formas de valorar

Tipos de valoraciones según la forma de llevarlas a cabo:

- Directas: Se valora el analito $\ce{A}$ con una disolución de $\ce{R}$, encontrándose el punto final utilizando el indicador adecuado.

- Indirectas:

$\ce{A + R_{auxiliar} -> P}$

$\ce{P + R -> P'\:}\llap{\boxed{\phantom{\ce{P + R -> P'}}}} \Rightarrow$ valoración que se realiza, según su resultado se obtiene la $[\ce{A}]$

- Por retroceso: Se adiciona sobre $\ce{A}$ un exceso exactamente conocido de $\ce{R}$.

$\ce{A + R -> P + R_{excedente}}$

A continuación se valora el exceso de $\ce{R}$, y con el resultado obtenido se encuentra la $[\ce{A}]$.

$\ce{R_{excedente} + R' -> P'}$
