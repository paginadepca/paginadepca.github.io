Ácido y base conjugados

  • Temario

$ \ce{ HA + H2O <=> A- \! + H3O+ } $

La pareja ácido-base $\ce{HA\,/\,A-} \!$ forman lo que se conoce como un par conjugado ácido-base. Así pues $\ce{A-} \!$ es la base conjugada del ácido $\ce{HA}$.

$ \ce{ B- \! + H2O <=> BH + OH- } $

De forma análoga, $\ce{BH}$ es el ácido conjugado de la base $\ce{B-} \!$.
