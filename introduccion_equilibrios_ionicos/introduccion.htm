Por definición, un equilibrio iónico es aquél en el que intervienen especies iónicas.

Son equilibrios iónicos homogéneos, tienen lugar en una única fase líquida, el ácido-base, el de complejación y el de oxidación-reducción (o redox). Es un equilibrio heterogéneo, involucra a una fase líquida y a otra sólida, el de precipitación.

Ejemplos de reacciones ácido-base:

$ \begin{array}{l} \ce{ HCl + NaOH -> NaCl + H2O } \\[1em] \ce{ HCl + NaHCO3 -> H2CO3 -> CO2 \! ^ \!\! + H2O } \\[1em] \ce{ 2 HCl + Mg(OH)2 -> MgCl2 + 2 H2O } \end{array} $

Ej. de complejación:

$ \begin{array}{l} \ce{ Fe^3+ \! + 6 CN- <=> [Fe(CN)6]^3- } \\[1em] \ce{ Fe^3+ \! + 6 CO <=> [Fe(CO)6]^3+ } \end{array} $

Ej. de oxidación-reducción:

$ \begin{array}{l} \ce{ 4 Fe^2+ \! + O2 + 4 H+ -> 4 Fe^3+ \! + 2 H2O } \\[1em] \ce{ Zn + 2 HCl -> ZnCl2 + H2 } \end{array} $

Ej. de precipitación:

$ \begin{array}{l} \ce{ Ba^2+ \! + SO4^2- -> BaSO4 \! v } \end{array} $
