Compuestos anfóteros y anfipróticos

  • Temario
  • Inicio

Un compuesto anfótero (o anfolito) es aquel que puede reaccionar tanto como ácido frente a bases o como base frente a ácidos.

$ \begin{array}{l} \ce{ Al(OH)3 + 3 H+ -> Al^{3+} \! + 3 H2O } \\[1ex] \ce{ Al(OH)3 + OH- -> AlO(OH)2^- \! + H2O } \end{array} $

Un compuesto anfiprótico es aquel que tanto puede ceder un protón como ganarlo.

$ \begin{array}{l} \begin{aligned} \ce{ HCO3^- \! + H2O -> CO3^{2-} \! + H3O+ } \end{aligned} \\[1ex] \begin{aligned} \ce{HCO3^- \! + H2O -> H2&CO3 + OH-} \\ &\, \smash{\raise 0.5ex \unicode{x21b3}} \: \ce{CO2 \! (^) \!\! + H2O} \end{aligned} \end{array} $

Por ejemplo, el agua es un solvente anfiprótico, como se observa en su reacción de autoprotólisis.

$ \ce{ 2 H2O <=> H3O+ \! + OH- } $
