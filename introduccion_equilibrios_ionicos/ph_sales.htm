  • $\ce{NaOH, KOH}$ ó $\ce{Ba(OH)2} \Rightarrow \ce{pH}$ básico

Son totalmente solubles, por tanto $\ce{pH}$ básico.

$\begin{array}{l}\ce{NaOH -> Na+ \! + OH-}\end{array}$

  • $\ce{KCl}$ ó $\ce{NaNO3} \Rightarrow \ce{pH}$ neutro

$ \begin{array}{l} \ce{ KCl -> K+ \! + Cl- } \\[1ex] \ce{ NaNO3 -> Na+ \! + NO3^- } \end{array} $

  • $\ce{Al(NO3)3}$ ó $\ce{FeCl3} \Rightarrow \ce{pH}$ ácido

$ \begin{array}{l} \ce{ Al(NO3)3 -> Al^3+ \! + 3 NO3^- } \\[1ex] \ce{ Al^3+ \! + 6 H2O -> [Al(H2O)6]^3+ \! <=> [Al(H2O)5OH]^2+ \! + H+ } \\[1em] \ce{ FeCl3 -> Fe^3+ \! + 3 Cl- } \\[1ex] \ce{ Fe^3+ \! + 6 H2O -> [Fe(H2O)6]^3+ \! <=> [Fe(H2O)5OH]^2+ \! + H+ } \end{array} $

  • $\ce{Na2CO3} \Rightarrow \ce{pH}$ básico

$ \begin{array}{l} \ce{ Na2CO3 -> 2 Na+ \! + CO3^2- } \\[1ex] \ce{ CO3^2- \! + H2O <=> HCO3^- \! + OH- } \end{array} $
